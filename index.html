Exercício de Formulário

Pressione Alt + tecla sublinhada para acessar os campos rapidamente. Use Tab para navegar entre os campos.

Nome deve conter apenas letras e ter no mínimo 3 caracteres
Insira um e-mail válido (exemplo: usuario@dominio.com)
Digite um CPF válido (ex: 123.456.789-00 ou 12345678900)
Tipo de Usuário *
Selecione um tipo de usuário
Digite um telefone válido (ex: (11) 98765-4321 ou 11987654321)
Insira uma data de nascimento válida
Insira uma URL válida começando com http:// ou https://
Selecione um estado
A senha deve conter pelo menos 8 caracteres, incluindo maiúsculas, minúsculas, números e caracteres especiais
As senhas devem ser iguais
Você deve aceitar os termos para continuar
